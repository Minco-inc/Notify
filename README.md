# Notify
Notify command for linux using Discord webhook!

# Installation
```bash
$ sudo ./ez-setup
```
And follow instruction on console.

# Usage
```bash
notify [message]
```

# More Help
```bash
make help
```
or
```bash
./bash/help
```
